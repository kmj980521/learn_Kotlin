## if~else를 표현식으로 사용하기 

 ```kotlin
  
  fun main(){

    val value : Int = if(10>5)
    {
        println("10은 5보다 크다")
        10
    }
    else
    {
        println("10은 5보다 크지 않다.")
        5
    }
    println(value) //10 
}
  
 ```
