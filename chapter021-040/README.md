## if~else 표현식의 타입
 
```kotlin
 
 fun main(){

    val value : Int = if(10>5)
    {
        println("10은 5보다 크다")
        10
    }
    else
    {
        println("10은 5보다 크지 않다.")
        5
    }
    println(value) //10
}

``` 
  
- if 블록과 else 블록의 마지막 표현식의 타입은 일치해야 한다 
- 만약 {} 블록이 비어있다면 Unit 타입이 되며 의미 없는 값이 저장된다 
- 이 표현식을 삼항 연산자처럼 사용 가능하므로 코틀린에서는 사함 연산자가 존재하지 않는다 
  
</details>
